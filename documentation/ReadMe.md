# Déplacement for chamber trio and EMG

## Atau Tanaka 2024

### Documentation folder

* Score (PDF, Sibelius)
* Dossier (PDF)
* Audio recording (Bandcamp)
* Video (YouTube)


#### Score
This folder contains initial pencil sketches for the piece, then the score in Sibelius, and the final performance version in PDF.

#### Dossier
The file, ["Tanaka Eramaa Deplacement.pdf"](https://github.com/ataut/deplacement-eramaa/blob/main/documentation/Tanaka%20Eramaa%20Deplacement.pdf) provides programme notes for the piece, a stage diagram and photos of EMG electrode placement, the EAVI EMG board, and the Bela Pepper in a Eurorack modular synth rack

#### Audio recording 
The recorded release of the piece on the Modulisme label on Bandcamp.

[modulisme.bandcamp.com/track/atau-tanaka-deplacement](https://modulisme.bandcamp.com/track/atau-tanaka-deplacement)
